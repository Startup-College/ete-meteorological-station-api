## Instalação  do Projeto

### Requisitos

- NodeJS => v16
- npm
- git

### Clonar repositório

```bash
 git clone https://github.com/Startup-College/ete-meteorological-station-api.git
```


### Instalar as dependências

```bash
  npm install
```


### gerar o modelo do banco

```bash
  npx prisma migrate dev
```

### Rodar o projeto

```bash
  npm run dev
```
